# Work
- 这是一个电脑运行，能够导入学生一个学期的时间表，创建一个新的时间表或任务或工作，通过设置重要程度和需要提醒日期给用户提醒和显示时间表内容
# INSTALL
-项目用c++书写，你可以选择某个模块书写；
基本要求是电脑上有c++的编译程序，推荐vs和sublime；
需要十分注意文件的命名：以所有字母都小写的主名称作为前缀，后跟一个破折号，再跟一个版本号，扩展说明，以及其他后缀。
